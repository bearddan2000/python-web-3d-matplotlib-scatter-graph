# python-web-3d-matplotlib-scatter-graph

## Description
A demo of creating a 3d scatter graph.

## Tech stack
- numpy
- matplotlib
- pandas

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`
[Availble here](http://localhost)

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
